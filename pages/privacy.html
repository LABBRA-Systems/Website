Last Updated: April 2026

This Privacy Policy explains what information is collected when you visit https://www.labbrasystems.com, how that information is used, and your rights regarding it. This Website is informational only and does not offer user accounts, payments, or interactive features.

Labbra Systems does not directly collect personal information from visitors.

  • IP addresses

This information is stored in GitHub's server logs and handled under GitHub's own privacy policy.

We do not collect or store:

  • Names
  • Email addresses
  • Phone numbers
  • Physical addresses
  • Payment information
  • Cookies or tracking identifiers
  • Behavioral analytics
  • User-generated content
  • Social media data
  • Device IDs
  • Location data beyond what any web server receives

2. How We Use Information

Labbra Systems does not process personal data.

Any technical information collected by GitHub may be used for:

  • Website security
  • Server performance monitoring
  • Basic operational diagnostics

We do not use this information for marketing, profiling, or analytics.

3. Data Retention

Labbra Systems does not store personal data on our servers. Any technical logs collected by GitHub may be retained by GitHub according to their policies. If we receive anonymized or technical data through hosting logs, it may be retained for up to 1 year for operational and security purposes.

4. Cookies and Tracking

This Website does not use:

No tracking technologies are deployed by Labbra Systems.

5. Third-Party Service Providers

Our Website is hosted by GitHub Pages, which may process technical data to operate the service. We do not share data with any other third parties.

For more information on GitHub's data practices, please refer to GitHub's privacy policy.

7. Data Sharing and Selling
